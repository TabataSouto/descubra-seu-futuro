# 🤖 IA Mentor de Carreira

Projeto desenvolvido para demonstrar como agentes de Inteligência Artificial podem auxiliar profissionais de tecnologia na descoberta de carreiras, planejamento de estudos e evolução profissional.

## 🎯 Objetivo

O IA Mentor de Carreira é um projeto que utiliza Inteligência Artificial para ajudar pessoas a descobrirem caminhos profissionais em tecnologia, identificando interesses, sugerindo áreas de atuação e indicando habilidades importantes para o desenvolvimento da carreira.

A proposta é simular uma mentoria profissional utilizando agentes especializados que trabalham em sequência para analisar o perfil do usuário e gerar um plano de evolução personalizado.

---

## 🚀 Como Funciona

O projeto é dividido em dois agentes principais.

### 1️⃣ Agente Entrevistador

O primeiro agente conduz uma entrevista estruturada para entender o perfil do usuário.

Durante a conversa são analisados:

- Experiência profissional
- Objetivos de carreira
- Disponibilidade para estudos
- Preferências de trabalho
- Interesses técnicos
- Motivações profissionais
- Experiências anteriores

Ao final, o agente sugere carreiras compatíveis com o perfil identificado.

### 2️⃣ Agente Planejador

Após a escolha da carreira, o segundo agente recebe as informações coletadas e cria um plano de desenvolvimento personalizado.

O plano inclui:

- Visão da profissão
- Mapa de habilidades
- Roadmap de estudos
- Projeto de portfólio
- Preparação para entrevistas
- Próximos passos para evolução profissional
